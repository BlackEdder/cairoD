class PSSurface: cairo.cairo.Surface;
The PostScript surface is used to render cairo graphics to Adobe PostScript files and is a multi-page vector surface backend.

this(cairo_surface_t* ptr);
Create a PSSurface from a existing cairo_surface_t*. PSSurface is a garbage collected class. It will call cairo_surface_destroy when it gets collected by the GC or when dispose() is called.

Warning:
ptr's reference count is not increased by this function! Adjust reference count before calling it if necessary

Only use this if you know what your doing! This function should not be needed for standard cairoD usage.

this(string fileName, double width, double height);
Creates a PostScript surface of the specified size in points to be written to filename.

Note that the size of individual pages of the PostScript output can vary. See setSize().

Parameters:
double width width of the surface, in points (1 point == 1/72.0 inch)
double height height of the surface, in points (1 point == 1/72.0 inch)

this(double width, double height);
Creates a PostScript surface of the specified size in points.This will generate a PSSurface that may be queried and used as a source, without generating a temporary file.

Parameters:
double width width of the surface, in points (1 point == 1/72.0 inch)
double height height of the surface, in points (1 point == 1/72.0 inch)

void restrictToLevel(PSLevel level);
Restricts the generated PostSript file to level. See getPSLevels() for a list of available level values that can be used here.

This function should only be called before any drawing operations have been performed on the given surface. The simplest way to do this is to call this function immediately after creating the surface.

void setEPS(bool eps);
If eps is true, the PostScript surface will output Encapsulated PostScript.

This function should only be called before any drawing operations have been performed on the current page. The simplest way to do this is to call this function immediately after creating the surface. An Encapsulated PostScript file should never contain more than one page.

bool getEPS(bool eps);
Check whether the PostScript surface will output Encapsulated PostScript.

void setSize(double width, double height);
hanges the size of a PostScript surface for the current (and subsequent) pages.

This function should only be called before any drawing operations have been performed on the current page. The simplest way to do this is to call this function immediately after creating the surface or immediately after completing a page with either Context.showPage() or Context.copyPage().

Parameters:
double width width of the surface, in points (1 point == 1/72.0 inch)
double height height of the surface, in points (1 point == 1/72.0 inch)

void dscBeginSetup();
This function indicates that subsequent calls to dscComment() should direct comments to the Setup section of the PostScript output.

This function should be called at most once per surface, and must be called before any call to dscBeginPageSetup() and before any drawing is performed to the surface.

See dscComment() for more details.

void dscBeginPageSetup();
This function indicates that subsequent calls to dscComment() should direct comments to the PageSetup section of the PostScript output.

This function call is only needed for the first page of a surface. It should be called after any call to dscBeginSetup() and before any drawing is performed to the surface.

See dscComment() for more details.
