Used to retrieve the list of supported versions. See PDFSurface.restrictToVersion().

string PDFVersionToString(PDFVersion vers);
Get the string representation of the given version id. This function will return null if version isn't valid. See getPDFVersions() for a way to get the list of valid version ids.

class PDFSurface: cairo.cairo.Surface;
The PDF surface is used to render cairo graphics to Adobe PDF files and is a multi-page vector surface backend.

this(cairo_surface_t* ptr);
Create a PDFSurface from a existing cairo_surface_t*. PDFSurface is a garbage collected class. It will call cairo_surface_destroy when it gets collected by the GC or when dispose() is called.

Warning:
ptr's reference count is not increased by this function! Adjust reference count before calling it if necessary

Only use this if you know what your doing! This function should not be needed for standard cairoD usage.

this(string fileName, double width, double height);
Creates a PDF surface of the specified size in points to be written to filename.

Parameters:
string fileName a filename for the PDF output (must be writable)
double width width of the surface, in points (1 point == 1/72.0 inch)
double height height of the surface, in points (1 point == 1/72.0 inch)

this(double width, double height);
Creates a PDF surface of the specified size in points. This will generate a PDF surface that may be queried and used as a source, without generating a temporary file.

Parameters:
double width width of the surface, in points (1 point == 1/72.0 inch)
double height height of the surface, in points (1 point == 1/72.0 inch)

void restrictToVersion(PDFVersion vers);
Restricts the generated PDF file to version. See getPDFVersions() for a list of available version values that can be used here.

This function should only be called before any drawing operations have been performed on the given surface. The simplest way to do this is to call this function immediately after creating the surface

void setSize(double width, double height);
Changes the size of a PDF surface for the current (and subsequent) pages.

This function should only be called before any drawing operations have been performed on the current page. The simplest way to do this is to call this function immediately after creating the surface or immediately after completing a page with either Context.showPage() or Context.copyPage().

Parameters:
double width width of the surface, in points (1 point == 1/72.0 inch)
double height height of the surface, in points (1 point == 1/72.0 inch)
